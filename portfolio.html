Portfolio

GitHub

Here are some of the projects I have worked on, sorted by order of relevance!

Pulsar

Personal Project

Pulsar is my main personal project idea that I have been working on since I was a freshman. It is a work-in-progress 3D space engine. Originally, it was implemented with Three.js, but now I am moving over to C++ with OpenGL.

Aggienova

STEM Research Leadership Program

Aggienova is a professor-led research project exploring the ultraviolet properties of supernovae. Most of my contributions to the project have been adding new entries to, organizing, and de-duplicating the database.

Virtual eSports Coach

Aggie Coding Club

Virtual eSports Coach was a web application that helps video gamers improve at video games. I was a co-project manager for this project, and it uses the MERN stack.

Bad Business

Chillenium Game Jam Entry

Bad Business was my team's entry to the Chillenium game jam in March 2024. It is a 2D dungeon crawler game where the player ventures into the monster-infested underground to find and sell random objects.

Unboringify

tidaltamu Hackathon Entry

Unboringify was my team's entry to the tidaltamu hackathon in November 2023. It is a machine learning-based project that improves PowerPoint presentations for teachers and professors.
